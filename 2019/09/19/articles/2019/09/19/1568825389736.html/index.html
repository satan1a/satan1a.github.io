一些步骤总结

第一次做PWN题，实际操作，感觉还是挺有趣的，虽然也有些套路的性质

  • file proc

    • 看程序的位数，是否为动态链接

  • chmod 777 proc & ./proc

    • 运行程序，观察程序的输入输出特征，以及字符串，便于定位
  • checksec proc
    • 看程序的保护机制，针对不同的保护机制设计不同的攻击思路和绕过方法

  • gdb ./proc

    • cyclic 100，直接start contin 输入，看返回地址泄漏

  • objdump -R ./proc

  • objdump -D ./proc

    • 看反汇编代码

  • IDA查看反汇编伪C代码
